<div style='position: absolute; right: 10px;'>Language:<select size=1 onChange="window.location.href='problem?id=1056&lang='+this.value+'&change=true'"><option value='default' selected>Default</option></select></div><div class="ptt" lang="en-US">IMMEDIATE DECODABILITY</div>
<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 10000K</td></tr><tr><td><b>Total Submissions:</b> 13062</td><td width="10px"></td><td><b>Accepted:</b> 6254</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US">An encoding of a set of symbols is said to be immediately decodable if no code for one symbol is the prefix of a code for another symbol.  We will assume for this problem that all codes are in binary, that no two codes within a set of codes are the same, that each code has at least one bit and no more than ten bits, and that each set has at least two codes and no more than eight.<br><br>Examples:  Assume an alphabet that has symbols {A, B, C, D}<br><br>The following code is immediately decodable:<br>     A:01   B:10   C:0010   D:0000<br><br>but this one is not:<br>     A:01   B:10   C:010   D:0000    (Note that A is a prefix of C)<br></div><p class="pst">Input</p><div class="ptx" lang="en-US">Write a program that accepts as input a series of groups of records from standard input.  Each record in a group contains a collection of zeroes and ones representing a binary code for a different symbol.  Each group is followed by a single separator record containing a single 9; the separator records are not part of the group. Each group is independent of other groups; the codes in one group are not related to codes in any other group (that is, each group is to be processed independently). </div><p class="pst">Output</p><div class="ptx" lang="en-US"> For each group, your program should determine whether the codes in that group are immediately decodable, and should print a single output line giving the group number and stating whether the group is, or is not, immediately decodable.</div><p class="pst">Sample Input</p><pre class="sio">01
10
0010
0000
9
01
10
010
0000
9
</pre><p class="pst">Sample Output</p><pre class="sio">Set 1 is immediately decodable
Set 2 is not immediately decodable
 
</pre><p class="pst">Source</p><div class="ptx" lang="en-US"><a href="searchproblem?field=source&key=Pacific+Northwest+1998">Pacific Northwest 1998</a></div></td></tr></table>
