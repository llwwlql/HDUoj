<div style='position: absolute; right: 10px;'>Language:<select size=1 onChange="window.location.href='problem?id=3787&lang='+this.value+'&change=true'"><option value='default' selected>Default</option></select></div><div class="ptt" lang="en-US">Convex Hull of Lattice Points</div>
<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 65536K</td></tr><tr><td><b>Total Submissions:</b> 485</td><td width="10px"></td><td><b>Accepted:</b> 296</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US">A <b>lattice point</b> is a point with <b>integer</b> coordinates.  A lattice polygon is a polygon with all vertices lattice points.  <br> <br>A polygon is <b>convex</b> if any line segment between two points of the polygon is inside (or on the boundary of) the polygon.   Equivalently, the interior angle at each polygon vertex is less than 180 degrees.  <br> <br>For a set S, of lattice points, the <b>convex hull</b> is the smallest convex (lattice) polygon which contains all points of the set.  (The vertices of the convex hull must be members of the set of lattice points).  If all the points are on a single straight line, the convex hull will be a line segment (a <b>degenerate</b> polygon – see rightmost diagram below).  In the diagrams below, the points of the set are indicated by solid dots, the vertices of the convex hull by X’s and the convex hull is drawn connecting the vertices. Note that not all points on the convex hull polygon are vertices. <br> <br><center><img src=images/3787_1.png></center><br> <br>The vertices of a lattice polygon are in standard order if: <br>a) The first vertex is the one with the largest y value.  If two vertices have the same y value, the one with the smaller x value is the first. <br>b) Vertices are given in clockwise order around the polygon. <br> <br>Write a program that reads a set of lattice points and outputs the vertices of the convex hull of the points in standard order. </div><p class="pst">Input</p><div class="ptx" lang="en-US">The first line of input contains a single integer P, (1 ≤ P ≤ 1000), which is the number of data sets that follow.  The first line of each data set contains the data set number, followed by a space, followed by a decimal integer giving the number of points N, (3 ≤ N ≤ 50), in the set.  The remaining lines in the data set contain the points in the set, at most 5 points per line (the last line may have fewer).  Each point consists of 2 space separated decimal integer values representing the x and y coordinates respectively.</div><p class="pst">Output</p><div class="ptx" lang="en-US">For each data set there are multiple lines of output.  The first line contains a decimal integer giving the data set number followed by a single space, followed by a decimal integer giving the total number of vertices of the convex hull.  The vertices of the convex hull follow, one per line in standard order. Each line contains the decimal integer x coordinate, a single space and the decimal integer y coordinate.</div><p class="pst">Sample Input</p><pre class="sio">4
