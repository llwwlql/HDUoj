<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 65536K</td></tr><tr><td><b>Total Submissions:</b> 9239</td><td width="10px"></td><td><b>Accepted:</b> 4865</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US">The expression N!, read as "N factorial," denotes the product of the first N positive integers, where N is nonnegative. So, for example, <br><pre> N       N! <br> 0       1 <br> 1       1 <br> 2       2 <br> 3       6 <br> 4      24 <br> 5     120 <br>10 3628800 </pre><br>For this problem, you are to write a program that can compute the last non-zero digit of any factorial for (0 <= N <= 10000). For example, if your program is asked to compute the last nonzero digit of 5!, your program should produce "2" because 5! = 120, and 2 is the last nonzero digit of 120. </div><p class="pst">Input</p><div class="ptx" lang="en-US">Input to the program is a series of nonnegative integers not exceeding 10000, each on its own line with no other letters, digits or spaces. For each integer N, you should read the value and compute the last nonzero digit of N!. </div><p class="pst">Output</p><div class="ptx" lang="en-US">For each integer input, the program should print exactly one line of output. Each line of output should contain the value N, right-justified in columns 1 through 5 with leading blanks, not leading zeroes. Columns 6 - 9 must contain " -> " (space hyphen greater space). Column 10 must contain the single last non-zero digit of N!. </div><p class="pst">Sample Input</p><pre class="sio">1
2
26
125
3125
9999</pre><p class="pst">Sample Output</p><pre class="sio">    1 -> 1
    2 -> 2
   26 -> 4
  125 -> 8
 3125 -> 2
 9999 -> 8</pre><p class="pst">Source</p><div class="ptx" lang="en-US"><a href="searchproblem?field=source&key=South+Central+USA+1997">South Central USA 1997</a></div></td></tr></table>
