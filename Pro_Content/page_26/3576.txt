<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 5000MS</td><td width="10px"></td><td colspan="3"><b>Memory Limit:</b> 65536K</td></tr><tr><td><b>Total Submissions:</b> 1869</td><td width="10px"></td><td><b>Accepted:</b> 497</td><td width="10px"></td><td style="font-weight:bold; color:red;">Special Judge</td></tr></table></div><p class="pst">Description</p><div class="ptx" lang="en-US"><p>Deterministic Final-State Automaton (DFA) is a directed multigraph whose vertices are called <i>states</i> and edges are called <i>transitions</i>. Each DFA transition is labeled with a single letter. Moreover, for each state <i>s</i> and each letter <i>l</i> there is at most one transition that leaves <i>s</i> and is labeled with <i>l</i>. DFA has a single <i>starting</i> state and a subset of <i>final states</i>. DFA defines a language of all words that can be constructed by writing down the letters on a path from the starting state to some final state.</p><p>Given a language with a finite set of words it is always possible to construct a DFA that defines this language. The picture on the left shows such DFA for the language consisting of three words: <tt>fix</tt>, <tt>foo</tt>, <tt>ox</tt>. However, this DFA has 7 states, which is not optimal. The DFA on the right defines the same language with just 5 states.</p><div align="center"><img src="images/3576_1.gif"></div><p>Your task is to find the minimum number of states in a DFA that defines the given language.</p></div><p class="pst">Input</p><div class="ptx" lang="en-US"><p>The first line of the input file contains a single integer number <i>n</i> (1 ≤ <i>n</i> ≤ <nobr>5 000</nobr>) — the number of words in the language. It is followed by <i>n</i> lines with a word on each line. Each word consists of 1 to 30 lowercase Latin letters from “<tt>a</tt>” to “<tt>z</tt>”. All words in the input file are different.</p></div><p class="pst">Output</p><div class="ptx" lang="en-US"><p>Write to the output file a single integer number — the minimal number of states in a DFA that defines the language from the input file.</p></div><p class="pst">Sample Input</p><pre class="sio"><table style="border-color: black; border-collapse: collapse;" border="1"><tr><th>#1</th><td>3<br>fix<br>foo<br>ox</td></tr><tr><th>#2</th><td>4<br>a<br>ab<br>ac<br>ad</td></tr></table></pre><p class="pst">Sample Output</p><pre class="sio"><table style="border-color: black; border-collapse: collapse;" border="1"><tr><th>#1</th><td>5</td></tr><tr><th>#2</th><td>3</td></tr></table></pre><p class="pst">Source</p><div class="ptx" lang="en-US"><a href="searchproblem?field=source&key=Northeastern+Europe+2007">Northeastern Europe 2007</a></div></td></tr></table>
