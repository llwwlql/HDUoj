<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td colspan="3"><b>Memory Limit:</b> 65536K</td></tr><tr><td><b>Total Submissions:</b> 185</td><td width="10px"></td><td><b>Accepted:</b> 37</td><td width="10px"></td><td style="font-weight:bold; color:red;">Special Judge</td></tr></table></div><p class="pst">Description</p><div class="ptx" lang="en-US"><p>Let number <b>A</b> be a <i>precise square </i>if there exists natural number <b>B</b> such that <b>B*B = A</b>.</p><p><b>2n</b>-digit number without leading zeroes is called a <i>supersquare</i> if it is a <i>precise square</i> and both <b>n</b>-digit numbers which are formed from its <b>n</b> first digits and its <b>n</b> last digits are <i>precise</i> <i>squares</i>. The second <b>n</b>-digit number d<sub>n</sub>d<sub>n-1</sub>…d<sub>1</sub> formed from <b>n</b> last digits may have leading zeroes but must not be equal zero.</p><div align="center"><img src="images/3560_1.gif"></div><p>You need to write a program which constructs a <b>2n</b>-digit number which is <i>supersquare</i>.</p></div><p class="pst">Input</p><div class="ptx" lang="en-US"><p>The input contains several test cases. The first line contains the number of test cases <b>T</b> (1 ≤ <b>T</b> ≤ 10). Each of the next <b>T</b> lines describes one test case and contains an integer number <b>n</b> (1 ≤ <b>n</b> ≤ 500).</p></div><p class="pst">Output</p><div class="ptx" lang="en-US"><p>The output consists of <b>T</b> lines, one line per each test case. Each line contains <b>2n</b>-digit <i>supersquare</i> number. If several solutions are possible only one of them should be given. When it is impossible to construct <b>2n</b>-digit <i>supersquare</i>, the line must contain <b>NO SUPERSQUARE POSSIBLE </b>phrase.</p></div><p class="pst">Sample Input</p><pre class="sio">2
1
2</pre><p class="pst">Sample Output</p><pre class="sio">49
1681</pre><p class="pst">Source</p><div class="ptx" lang="en-US"><a href="searchproblem?field=source&key=Northeastern+Europe+2005">Northeastern Europe 2005</a>, Western Subregion</div></td></tr></table>
