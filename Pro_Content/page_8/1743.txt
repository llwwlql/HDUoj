<div style='position: absolute; right: 10px;'>Language:<select size=1 onChange="window.location.href='problem?id=1743&lang='+this.value+'&change=true'"><option value='default' selected>Default</option></select></div><div class="ptt" lang="en-US">Musical Theme</div>
<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 30000K</td></tr><tr><td><b>Total Submissions:</b> 26840</td><td width="10px"></td><td><b>Accepted:</b> 9056</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US">A musical melody is represented as a sequence of N (1<=N<=20000)notes that are integers in the range 1..88, each representing a key on the piano. It is unfortunate but true that this representation of melodies ignores the notion of musical timing; but, this programming task is about notes and not timings. <br>Many composers structure their music around a repeating &qout;theme&qout;, which, being a subsequence of an entire melody, is a sequence of integers in our representation. A subsequence of a melody is a theme if it: <br><ul><li>is at least five notes long <br><li>appears (potentially transposed -- see below) again somewhere else in the piece of music <br><li>is disjoint from (i.e., non-overlapping with) at least one of its other appearance(s) </ul><br>Transposed means that a constant positive or negative value is added to every note value in the theme subsequence. <br>Given a melody, compute the length (number of notes) of the longest theme. <br>One second time limit for this problem's solutions! <br></div><p class="pst">Input</p><div class="ptx" lang="en-US">The input contains several test cases. The first line of each test case contains the integer N. The following n integers represent the sequence of notes.<br>The last test case is followed by one zero. <br></div><p class="pst">Output</p><div class="ptx" lang="en-US">For each test case, the output file should contain a single line with a single integer that represents the length of the longest theme. If there are no themes, output 0. </div><p class="pst">Sample Input</p><pre class="sio">30
25 27 30 34 39 45 52 60 69 79 69 60 52 45 39 34 30 26 22 18
82 78 74 70 66 67 64 60 65 80
0
</pre><p class="pst">Sample Output</p><pre class="sio">5</pre><p class="pst">Hint</p><div class="ptx" lang="en-US">Use scanf instead of cin to reduce the read time.</div><p class="pst">Source</p><div class="ptx" lang="en-US"><a href="searchproblem?field=source&key=LouTiancheng%40POJ">LouTiancheng@POJ</a></div></td></tr></table>
