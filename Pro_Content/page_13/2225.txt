<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 65536K</td></tr><tr><td><b>Total Submissions:</b> 3328</td><td width="10px"></td><td><b>Accepted:</b> 1237</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US">You're in space.<br>You want to get home.<br>There are asteroids.<br>You don't want to hit them. </div><p class="pst">Input</p><div class="ptx" lang="en-US">Input to this problem will consist of a (non-empty) series of up to 100 data sets. Each data set will be formatted according to the following description, and there will be no blank lines separating data sets. <br><br>A single data set has 5 components: <br><ol><li>Start line - A single line, "START N", where 1 <= N <= 10. <br><li>Slice list - A series of N slices. Each slice is an N x N matrix representing a horizontal slice through the asteroid field. Each position in the matrix will be one of two values: <br><ul>'O' - (the letter "oh") Empty space<br>'X' - (upper-case) Asteroid present </ul><br><li>Starting Position - A single line, "A B C", denoting the [A,B,C] coordinates of your craft's starting position. The coordinate values will be integers separated by individual spaces. <br><li>Target Position - A single line, "D E F", denoting the [D,E,F] coordinates of your target's position. The coordinate values will be integers separated by individual spaces. <br><li>End line - A single line, "END" </ol><br>The origin of the coordinate system is [0,0,0]. Therefore, each component of each coordinate vector will be an integer between 0 and N-1, inclusive. <br><br>The first coordinate in a set indicates the column. Left column = 0.<br>The second coordinate in a set indicates the row. Top row = 0.<br>The third coordinate in a set indicates the slice. First slice = 0. <br><br>Both the Starting Position and the Target Position will be in empty space. <br><br></div><p class="pst">Output</p><div class="ptx" lang="en-US">For each data set, there will be exactly one output set, and there will be no blank lines separating output sets. <br><br>A single output set consists of a single line. If a route exists, the line will be in the format "X Y", where X is the same as N from the corresponding input data set and Y is the least number of moves necessary to get your ship from the starting position to the target position. If there is no route from the starting position to the target position, the line will be "NO ROUTE" instead. <br><br>A move can only be in one of the six basic directions: up, down, left, right, forward, back. Phrased more precisely, a move will either increment or decrement a single component of your current position vector by 1. <br></div><p class="pst">Sample Input</p><pre class="sio">START 1
