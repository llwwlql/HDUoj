<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 65536K</td></tr><tr><td><b>Total Submissions:</b> 1195</td><td width="10px"></td><td><b>Accepted:</b> 784</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US">Within Settlers of Catan, the 1995 German game of the year, players attempt to dominate an island by building roads, settlements and cities across its uncharted wilderness. <br>You are employed by a software company that just has decided to develop a computer version of this game, and you are chosen to implement one of the game's special rules: <br><br>When the game ends, the player who built the longest road gains two extra victory points. <br>The problem here is that the players usually build complex road networks and not just one linear path. Therefore, determining the longest road is not trivial (although human players usually see it immediately). <br>Compared to the original game, we will solve a simplified problem here: You are given a set of nodes (cities) and a set of edges (road segments) of length 1 connecting the nodes. <br>The longest road is defined as the longest path within the network that doesn't use an edge twice. Nodes may be visited more than once, though. <br><br>Example: The following network contains a road of length 12. <br><pre><br>o      o--o      o<br> \    /    \    /<br>  o--o      o--o<br> /    \    /    \<br>o      o--o      o--o<br>           \    /<br>            o--o<br></pre></div><p class="pst">Input</p><div class="ptx" lang="en-US">The input will contain one or more test cases. <br>The first line of each test case contains two integers: the number of nodes n (2<=n<=25) and the number of edges m (1<=m<=25). The next m lines describe the m edges. Each edge is given by the numbers of the two nodes connected by it. Nodes are numbered from 0 to n-1. Edges are undirected. Nodes have degrees of three or less. The network is not neccessarily connected. <br>Input will be terminated by two values of 0 for n and m. </div><p class="pst">Output</p><div class="ptx" lang="en-US">For each test case, print the length of the longest road on a single line.</div><p class="pst">Sample Input</p><pre class="sio">3 2
0 1
1 2
15 16

1 2

3 4
3 5

10 12

10 13
12 14
0 0
</pre><p class="pst">Sample Output</p><pre class="sio">2
12
