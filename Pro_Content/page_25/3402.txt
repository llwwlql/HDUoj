<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 65536K</td></tr><tr><td><b>Total Submissions:</b> 876</td><td width="10px"></td><td><b>Accepted:</b> 246</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US"><div><p><i>K</i> chess queens (0 &le; <i>K</i> &lt; <i>M</i>&times;<i>N</i>) are located on a rectangular chessboard of size <i>M</i> &times; <i>N</i> (1 &le; <i>M</i>, <i>N</i> &le; 26) and there is no more than one queen in each cell. We consider a vacant cell is under attack if it's possible to move into it at least one of the queens with just one move according to the chess rules.</p><p>The task is to add one more queen into one of the vacant chessboard cells so that the number of cells under action will be minimal.</p><p>The chessboard rows are labeled with lowercase Latin letters starting with a from the bottom; the columns are labeled with numbers starting with 1 from the left. Thus the cells can be labeled as <i>a1</i>, <i>f23</i>, etc.</p></div></div><p class="pst">Input</p><div class="ptx" lang="en-US"><p>The input consists of <i>K</i>+1 lines: the first line contains the values <i>M</i>, <i>N</i> and <i>K</i> separated by one or several spaces; the next lines contain the labels of the cells where the queens are placed initially (one label per line).</p></div><p class="pst">Output</p><div class="ptx" lang="en-US"><p>The output contains two lines. The first line contains the label of the cell where the queen should be places. (If there are several possible answers, pick the cell that is first in the lexicographic ordering of the cell labels). The second one contains the number of empty cells not under attack after the queen has been added.</p></div><p class="pst">Sample Input</p><pre class="sio">4 4 2
a1
a2</pre><p class="pst">Sample Output</p><pre class="sio">c1
2</pre><p class="pst">Source</p><div class="ptx" lang="en-US"><a href="searchproblem?field=source&key=Northeastern+Europe+2001">Northeastern Europe 2001</a>, Western Subregion</div></td></tr></table>
