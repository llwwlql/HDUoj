<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 131072K</td></tr><tr><td><b>Total Submissions:</b> 3090</td><td width="10px"></td><td><b>Accepted:</b> 1187</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US"><p>An image <i>A</i> is said to be a subimage of another image <i>B</i> if it is possible to remove some rows and/or columns of pixels from <i>B</i> so that the resulting image is identical to <i>A</i>. Figure 6 illustrates an example. Image <i>A</i>, shown in Figure 6(a), is a subimage of image <i>B</i>, shown in Figure 6(b), because the image resulting from the removal of the middle row and column of pixels from <i>B</i> is identical to <i>A</i>.</p><div align="center"><table><tr><td align="center"><table style=" border-collapse: collapse;"><tr><td style="width: 1em; height: 1em; border: solid black 1pt;" bgcolor="black">&nbsp;</td><td style="width: 1em; height: 1em; border: solid black 1pt;" bgcolor="black">&nbsp;</td></tr><tr><td style="width: 1em; height: 1em; border: solid black 1pt;" bgcolor="black">&nbsp;</td><td style="width: 1em; height: 1em; border: solid black 1pt;">&nbsp;</td></tr></table></td><td style="width: 36pt"></td><td align="center"><table style="border-collapse: collapse"><tr><td style="width: 1em; height: 1em; border: solid black 1pt;" bgcolor="black">&nbsp;</td><td style="width: 1em; height: 1em; border: solid black 1pt;">&nbsp;</td><td style="width: 1em; height: 1em; border: solid black 1pt;" bgcolor="black">&nbsp;</td></tr><tr><td style="width: 1em; height: 1em; border: solid black 1pt;">&nbsp;</td><td style="width: 1em; height: 1em; border: solid black 1pt;">&nbsp;</td><td style="width: 1em; height: 1em; border: solid black 1pt;">&nbsp;</td></tr><tr><td style="width: 1em; height: 1em; border: solid black 1pt;" bgcolor="black">&nbsp;</td><td style="width: 1em; height: 1em; border: solid black 1pt;">&nbsp;</td><td style="width: 1em; height: 1em; border: solid black 1pt;">&nbsp;</td></tr></table></td></tr><tr style="font-size: 10pt"><td align="center">(a) Image <i>A</i></td><td style="width: 36pt"></td><td align="center">(b) Image <i>B</i></td></tr></table></div><p align="center">Figure 6: An example of a subimage</p><p>Given two black-and-white images <i>A</i> and <i>B</i>, determine whether <i>A</i> is a subimage of <i>B</i>.</p></div><p class="pst">Input</p><div class="ptx" lang="en-US"><p>The input contains a single test case. The first line contains two integers <i>r</i> and <i>c</i> (1 ≤ <i>r</i>, <i>c</i> ≤ 20), the dimensions of <i>A</i>. The following r lines, each containing a string of length <i>c</i>, give an <i>r</i> × <i>c</i> 0-1 matrix representing the pixels of <i>A</i>. The next line contains two integers <i>R</i> and <i>C</i> (<i>r</i> ≤ <i>R</i> ≤ 20; <i>c</i> ≤ <i>C</i> ≤ 20), the dimensions of <i>B</i>. The following <i>R</i> lines, each containing a string of length <i>C</i>, give an <i>R</i> × <i>C</i> 0-1 matrix representing the pixels of <i>B</i>. A 0 indicates a white pixel; a 1 indicates a black pixel.</p></div><p class="pst">Output</p><div class="ptx" lang="en-US"><p>If <i>A</i> is a subimage of <i>B</i>, print “<tt>Yes</tt>”; otherwise, print “<tt>No</tt>”.</p></div><p class="pst">Sample Input</p><pre class="sio">2 2
11
10
3 3
101
000
100</pre><p class="pst">Sample Output</p><pre class="sio">Yes</pre><p class="pst">Source</p><div class="ptx" lang="en-US"><a href="searchproblem?field=source&key=PKU+Campus+2008+%28POJ+Founder+Monthly+Contest+%E2%80%93+2008.05.10%29">PKU Campus 2008 (POJ Founder Monthly Contest – 2008.05.10)</a>, frkstyc</div></td></tr></table>
