<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 65536K</td></tr><tr><td><b>Total Submissions:</b> 2349</td><td width="10px"></td><td><b>Accepted:</b> 584</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US">It is a historical fact that during the legendary voyage of "Titanic" the wireless telegraph machine had delivered 6 warnings about the danger of icebergs. Each of the telegraph messages described the point where an iceberg had been noticed. The first five warnings were transferred to the captain of the ship. The sixth one came late at night and a telegraph operator did not notice that the coordinates mentioned were very close to the current ship's position. <br><br>Write a program that will warn the operator about the danger of icebergs! </div><p class="pst">Input</p><div class="ptx" lang="en-US">The input messages are of the following format: <br><pre>Message #&lt;n>.<br>Received at &lt;HH>:&lt;MM>:&lt;SS>. <br>Current ship's coordinates are <br>&lt;X1>^&lt;X2>'&lt;X3>" &lt;NL/SL> <br>and &lt;Y1>^&lt;Y2>'&lt;Y3>" &lt;EL/WL>.<br>An iceberg was noticed at <br>&lt;A1>^&lt;A2>'&lt;A3>" &lt;NL/SL> <br>and &lt;B1>^&lt;B2>'&lt;B3>" &lt;EL/WL>.<br>===</pre><br>Here &lt;n> is a positive integer, &lt;HH>:&lt;MM>:&lt;SS> is the time of the message reception, &lt;X1>^&lt;X2>'&lt;X3>" &lt;NL/SL> and &lt;Y1>^&lt;Y2>'&lt;Y3>" &lt;EL/WL> means "X1 degrees X2 minutes X3 seconds of North (South) latitude and Y1 degrees Y2 minutes Y3 seconds of East (West) longitude."</div><p class="pst">Output</p><div class="ptx" lang="en-US">Your program should print to the output file message in the following format:<br><pre>The distance to the iceberg: &lt;s> miles.</pre><br>Where &lt;s>  should be the distance between the ship and the iceberg, (that is the length of the shortest path  on the sphere between the ship and the iceberg). This distance should be printed up to (and correct to) two decimal digits. If this distance is less than (but not equal to!) 100 miles the program should print one more line with the text: <br><pre>DANGER!</pre></div><p class="pst">Sample Input</p><pre class="sio">Message #513.
Received at 22:30:11. 
Current ship's coordinates are 
41^46'00" NL 
and 50^14'00" WL.
An iceberg was noticed at
41^14'11" NL 
and 51^09'00" WL.
===
</pre><p class="pst">Sample Output</p><pre class="sio">The distance to the iceberg: 52.04 miles.
DANGER!
</pre><p class="pst">Hint</p><div class="ptx" lang="en-US">For simplicity of calculations  assume that the Earth is an ideal sphere with the diameter of 6875 miles completely covered with water. Also you can be sure that lines in the input file break exactly as it is shown in the input samples. The ranges of the ship and the iceberg coordinates are the same as the usual range for geographical coordinates, i.e. from 0 to 90 degrees inclusively for NL/SL and from 0 to 180 degrees inclusively for EL/WL.</div><p class="pst">Source</p><div class="ptx" lang="en-US"><a href="searchproblem?field=source&key=Ural+Collegiate+Programming+Contest+1999">Ural Collegiate Programming Contest 1999</a></div></td></tr></table>
