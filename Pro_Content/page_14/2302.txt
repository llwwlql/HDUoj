<div style='position: absolute; right: 10px;'>Language:<select size=1 onChange="window.location.href='problem?id=2302&lang='+this.value+'&change=true'"><option value='default' selected>Default</option></select></div><div class="ptt" lang="en-US">Traditional BINGO</div>
<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 65536K</td></tr><tr><td><b>Total Submissions:</b> 1110</td><td width="10px"></td><td><b>Accepted:</b> 548</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US">Traditional BINGO is played in person in a large hall. Players meet at the hall, pay a fee to get in, then the games begin. A night of BINGO consists of many BINGO games played continuously, one after another. <br>A single BINGO game proceeds like this: Each player has a number of BINGO cards (players can usually play any number of cards). Each BINGO card has 5 rows and 5 columns thus providing 25 spaces. <br><br>The columns are labeled from left to right with the letters: 'B', 'I', 'N', 'G', 'O'. With one exception (the center space is "free") the spaces in the card are assigned values as follows: <br><ul><li>Each space in the 'B' column contains a number from 1 - 15. <br><li>Each space in the 'I' column contains a number from 16 - 30. <br><li>Each space in the 'N' column contains a number from 31 - 45. <br><li>Each space in the 'G' column contains a number from 46 - 60. <br><li>Each space in the 'O' column contains a number from 61 - 75. </ul><br>Furthermore, a number can appear only once on a single card. <br>Here's a sample BINGO card:<br><TABLE cellPadding=5 border=1><TBODY><TR align=middle><TD><FONT size=5>B</FONT></TD><TD><FONT size=5>I</FONT></TD><TD><FONT size=5>N</FONT></TD><TD><FONT size=5>G</FONT></TD><TD><FONT size=5>O</FONT></TD></TR><TR align=middle><TD>10</TD><TD>17</TD><TD>39</TD><TD>49</TD><TD>64</TD></TR><TR align=middle><TD>12</TD><TD>21</TD><TD>36</TD><TD>55</TD><TD>62</TD></TR><TR align=middle><TD>14</TD><TD>25</TD><TD><FONT size=1>FREE<BR>SPACE</FONT></TD><TD>52</TD><TD>70</TD></TR><TR align=middle><TD>7</TD><TD>19</TD><TD>32</TD><TD>56</TD><TD>68</TD></TR><TR align=middle><TD>5</TD><TD>24</TD><TD>34</TD><TD>54</TD><TD>71</TD></TR></TBODY></TABLE><br>The number of unique BINGO cards is very large and can be calculated with this equation: <br><pre>// the B, I, G, and O columns * the N column<br>(15 * 14 * 13 * 12 * 11) ^ 4 * (15 * 14 * 13 * 12)</pre><br>While perhaps interesting to a statistician, the number of possible BINGO cards has nothing to do with player's chances of winning. <br>You will note that there are 75 possible BINGO numbers: <br><pre>B1, B2, B3, ... B15, I16, I17, I18, ... I30, N31, N32, ... O74, O75.</pre><br>Each of these numbers is represented by a ball in a large rotating bin. Each ball is painted with its unique BINGO number. An announcer spins the bin, reaches in a selects a ball, and a announces it to the room. The players check all of their cards to see if that number appears on their card. If it is, they mark it. A player may mark the centre FREE SPACE at any time. <br>When a player has a BINGO (5 marks in a row, column, or diagonal), he or she calls out BINGO. The game pauses while the card is verified. If indeed a winner, the game stops and a new game begins. If the card wasn't a winner, the game proceeds where it left off. Each BINGO game proceeds until someone wins (there's always a winner). </div><p class="pst">Input</p><div class="ptx" lang="en-US">The first line of input contains n, the number of BINGO games that you will analyze. n game descriptions follow. Each game description specifies a card to be played followed by a sequence of BINGO numbers. You are to determine, when the holder of the card will win the game, assuming the player has just this one card and there are no other players. <br><br>Each card description consists of five lines, giving the numbers on the card row by row. All but the 3rd row contain 5 numbers; the 3rd contains 4 because of the free space. One or more lines follow that represent some ordering of all 75 bingo numbers. All bingo numbers are simply integers between 1 and 75 - the one-letter prefix is redundant.<br></div><p class="pst">Output</p><div class="ptx" lang="en-US">For each game, ouput the line "BINGO after n numbers announced" as appropriate. <br><br><b>Chances of Winning</b><br><br>Every BINGO game has a winning card, so a player's chances of winning depend on the number of cards in the game and how many cards s/he is playing. For example, if a player has 12 cards in a game with 1200 cards, the chances of winning for that player is 1 in 100. </div><p class="pst">Sample Input</p><pre class="sio">1
