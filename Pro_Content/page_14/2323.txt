<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 5000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 65536K</td></tr><tr><td><b>Total Submissions:</b> 1338</td><td width="10px"></td><td><b>Accepted:</b> 668</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US">Count the number of permutations that have a specific number of inversions.<br>Given a permutation a1, a2, a3,..., an of the n integers 1, 2, 3, ..., n, an inversion is a pair (ai, aj) where i < j and ai > aj. The number of inversions in a permutation gives an indication on how "unsorted" a permutation is. If we wish to analyze the average running time of a sorting algorithm, it is often useful to know how many permutations of n objects will have a certain number of inversions.<br><br>In this problem you are asked to compute the number of permutations of n values that have exactly k inversions.<br><br>For example, if n = 3, there are 6 permutations with the indicated inversions as follows:<br><blockquote><pre>123			0 inversions<br> <br>132			1 inversion (3 > 2)<br> <br>213			1 inversion (2 > 1)<br> <br>231			2 inversions (2 > 1, 3 > 1)<br> <br>312			2 inversions (3 > 1, 3 > 2)<br> <br>321			3 inversions (3 > 2, 3 > 1, 2 > 1)</pre></blockquote> <br>Therefore, for the permutations of 3 things<br><ul><li>1 of them has 0 inversions <br><li>2 of them have 1 inversion <br><li>2 of them have 2 inversions <br><li>1 of them has 3 inversions <br><li>0 of them have 4 inversions <br><li>0 of them have 5 inversions <br><li>etc.</ul><br><br></div><p class="pst">Input</p><div class="ptx" lang="en-US">The input consists one or more problems. The input for each problem is specified on a single line, giving the integer n (1 <= n <= 18) and a non-negative integer k (0 <= k <= 200). The end of input is specified by a line with n = k = 0.</div><p class="pst">Output</p><div class="ptx" lang="en-US">For each problem, output the number of permutations of {1, ..., n}with exactly k inversions.</div><p class="pst">Sample Input</p><pre class="sio">3 0
3 1
3 2
3 3
4 2
4 10
13 23
18 80
0 0
</pre><p class="pst">Sample Output</p><pre class="sio">1
2
2
1
5
0
46936280
184348859235088
</pre><p class="pst">Source</p><div class="ptx" lang="en-US"><a href="searchproblem?field=source&key=Rocky+Mountain+2003">Rocky Mountain 2003</a></div></td></tr></table>
