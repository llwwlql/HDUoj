<div style='position: absolute; right: 10px;'>Language:<select size=1 onChange="window.location.href='problem?id=2445&lang='+this.value+'&change=true'"><option value='default' selected>Default</option></select></div><div class="ptt" lang="en-US">Squares</div>
<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 15000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 131072K</td></tr><tr><td><b>Total Submissions:</b> 892</td><td width="10px"></td><td><b>Accepted:</b> 321</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US">A children's board game consists of a square array of dots that contains lines connecting some dots. One part of the game requires that the players count the number of squares that are formed by these lines. For example, in the figure shown below, there are 3 squares, 2 of size 1 and 1 of size 2 , so the total number of squares is 3. (The "size" of a square is the number of lines segments required to form a side.) <br><center><img src=images/2445_1.jpg></center><br>Your task is to write a program to count the number of all the possible squares. </div><p class="pst">Input</p><div class="ptx" lang="en-US">The input represents a series of game boards. Each board consists of a description of a square array of n ^ 2 dots (where 2 <= n <= 1500) and some interconnecting horizontal and vertical lines. A record of a single board with n ^ 2 dots and m (m <= 300000 )interconnecting lines is formatted as follows: <br><br>Line 1: "n" the number of dots in a single row or column.<br>Line 2: "m" the number of interconnecting lines.<br>Each of the next m lines are one of following two types:<br>		<br>"H i j k" (1 <= i, j <= n, k >0, j + k <= n) indicates a horizontal line of length k from the dot in row i, column j to the dot in row i, column j + k.<br><br>or<br><br>"V i j k" (1 <= i, j <= n, k >0, i + k <= n) indicates a vertical line of length k from the dot in row i, column j to the dot in row i + k, column j.<br><br>The end of input is indicated by end-of-file. </div><p class="pst">Output</p><div class="ptx" lang="en-US">For each record print only one integer, which is the number of squares.</div><p class="pst">Sample Input</p><pre class="sio">4
9
H 1 1 1
H 1 3 1
H 2 1 3
H 3 2 1
H 4 2 2
V 1 1 1
V 1 2 3
V 2 3 1
V 1 4 3
4
9
H 1 1 1
H 1 3 1
H 2 1 3
H 3 2 1
H 4 2 2
V 1 1 1
V 1 2 3
V 2 3 1
V 1 4 3
</pre><p class="pst">Sample Output</p><pre class="sio">3
3
</pre><p class="pst">Source</p><div class="ptx" lang="en-US"><a href="searchproblem?field=source&key=POJ+Monthly">POJ Monthly</a>,Guang Lin</div></td></tr></table>
