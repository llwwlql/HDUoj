<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 65536K</td></tr><tr><td><b>Total Submissions:</b> 1526</td><td width="10px"></td><td><b>Accepted:</b> 577</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US">Surely you know someone who thinks he is very clever. You decide to let him down with the following problem:<br><ul><li>"Can you tell me what the syntax for a set is?", you ask him. <br><li>"Sure!", he replies, "a set encloses a possibly empty list of elements within two curly braces. Each element is either another set or a letter of the given alphabet. Elements in a list are separated by a comma." <br><li>"So if I give you a word, can you tell me if it is a syntactically correct representation of a set?" <br><li>"Of course, any fool can do it!" is his answer. </ul><br>Now you got him! You present him with the following grammar, defining formally the syntax for a set (which was described informally by him): <br><PRE>Set          ::= "{" Elementlist "}"<br>Elementlist  ::= &lt;empty&gt; | List<br>List         ::= Element | Element "," List<br>Element      ::= Atom | Set<br>Atom         ::= "{" | "}" | ","<br></PRE><br><SAMP>&lt;empty&gt;</SAMP> stands for the empty word, i.e., the list in a set can be empty.<br>Soon he realizes that this task is much harder than he has thought, since the alphabet consists of the characters which are also used for the syntax of the set. So he claims that it is not possible to decide efficiently if a word consisting of "{", "}" and "," is a syntactically correct representation of a set or not.<br>To disprove him, you need to write an efficient program that will decide this problem. </div><p class="pst">Input</p><div class="ptx" lang="en-US">The first line of the input contains a number representing the number of lines to follow. <br>Each line consists of a word, for which your program has to decide if it is a syntactically correct representation of a set. You may assume that each word consists of between 1 and 200 characters from the set { "{", "}", "," }. </div><p class="pst">Output</p><div class="ptx" lang="en-US">Output for each test case whether the word is a set or not. Adhere to the format shown in the sample output.</div><p class="pst">Sample Input</p><pre class="sio">4
{}
{{}}
{{}},{,}}
{,,}
</pre><p class="pst">Sample Output</p><pre class="sio">Word #1: Set
Word #2: Set
Word #3: Set
Word #4: No Set
</pre><p class="pst">Source</p><div class="ptx" lang="en-US"><a href="searchproblem?field=source&key=Ulm+Local+2005">Ulm Local 2005</a></div></td></tr></table>
