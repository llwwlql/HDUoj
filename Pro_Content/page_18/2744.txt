<div style='position: absolute; right: 10px;'>Language:<select size=1 onChange="window.location.href='problem?id=2744&lang='+this.value+'&change=true'"><option value='default' selected>Default</option></select></div><div class="ptt" lang="en-US">Atomic Car Race</div>
<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td colspan="3"><b>Memory Limit:</b> 65536K</td></tr><tr><td><b>Total Submissions:</b> 1216</td><td width="10px"></td><td><b>Accepted:</b> 587</td><td width="10px"></td><td style="font-weight:bold; color:red;">Special Judge</td></tr></table></div><p class="pst">Description</p><div class="ptx" lang="en-US">In the year 2020, a race of atomically energized cars will be held. Unlike today's car races, fueling is not a concern of racing teams. Cars can run throughout the course without any refueling. Instead, the critical factor is tire (tyre). Teams should carefully plan where to change tires of their cars.<br>The race is a road race having n checkpoints in the course. Their distances from the start are a1, a2, ..., and an (in kilometers). The n-th checkpoint is the goal. At the i-th checkpoint (i < n), tires of a car can be changed. Of course, a team can choose whether to change or not to change tires at each checkpoint. It takes b seconds to change tires (including overhead for braking and accelerating). There is no time loss at a checkpoint if a team chooses not to change tires.<br>A car cannot run fast for a while after a tire change, because the temperature of tires is lower than the designed optimum. After running long without any tire changes, on the other hand, a car cannot run fast because worn tires cannot grip the road surface well. The time to run an interval of one kilometer from x to x + 1 is given by the following expression (in seconds). Here x is a nonnegative integer denoting the distance (in kilometers) from the latest checkpoint where tires are changed (or the start). r, v, e and f are given constants.<br><ul>1/(v - e * (x - r)) (if x >= r)<br>1/(v - f * (r - x)) (if x < r)</ul><br>Your mission is to write a program to determine the best strategy of tire changes which minimizes the total time to the goal.</div><p class="pst">Input</p><div class="ptx" lang="en-US">The input consists of multiple datasets each corresponding to a race situation. The format of a<br>dataset is as follows.<br>n<br>a1 a2 . . . an<br>b<br>r v e f<br>The meaning of each of the input items is given in the problem statement. If an input line contains two or more input items, they are separated by a space.<br>n is a positive integer not exceeding 100. Each of a1, a2, ..., and an is a positive integer satisfying 0 < a1 < a2 < . . . < an <= 10000. b is a positive decimal fraction not exceeding 100.0. r is a nonnegative integer satisfying 0 <= r <= an - 1. Each of v, e and f is a positive decimal fraction. You can assume that v - e * (an - 1 - r) >= 0.01 and v - f * r >= 0.01. <br>The end of the input is indicated by a line with a single zero.</div><p class="pst">Output</p><div class="ptx" lang="en-US">For each dataset in the input, one line containing a decimal fraction should be output. The decimal fraction should give the elapsed time at the goal (in seconds) when the best strategy is taken. An output line should not contain extra characters such as spaces.<br>The answer should not have an error greater than 0.001. You may output any number of digits after the decimal point, provided that the above accuracy condition is satisfied.</div><p class="pst">Sample Input</p><pre class="sio">2
