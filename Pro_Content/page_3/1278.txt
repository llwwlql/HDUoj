<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 10000K</td></tr><tr><td><b>Total Submissions:</b> 418</td><td width="10px"></td><td><b>Accepted:</b> 136</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US">You are the owner of a very nice boat. You have many requests to rent your beautiful boat during the summer time and you decided to maximize your profit. For each client you know the number of days he wants to rent the boat and a list of choices, where a choice means an amount of money and a deadline. The deadline is the number of days counted from a time origin. You get the money for a given deadline (when the holiday must end) only if you are able to rent the boat before that deadline.<br>For example your friend Jack wants to travel on the Mediterranean Sea for 20 days and has the following choices:<br>-	if the holiday deadline is 60 days you can get 1000 EUR provided the boat is rented in between 0 and 41 (60days-20days+1) for 20 days;<br>-	if the holiday deadline is 70 days you can get 800 EUR.<br><br>If a client does not contribute to the maximization of your profit you can drop that client. You know all the clients and their choices and you must rent the boat according to the order in which the clients submit their requests.<br><br>Write a program that, given the clients, computes the largest profit you can get. <br></div><p class="pst">Input</p><div class="ptx" lang="en-US">The program input is from the standard input. Each data set in the input has the following format:<br><ul><br><li>	n - the number of clients (n <= 100)<br><li>	on n separate lines the number of days (at most 100) each client wishes to rent the boat;<br><li>	the total number of choices for all clients;<br><li>	the list of choices in the format client_id, deadline, amount_of_money where the client id is a positive integer in the range 1..n and deadline <= 100.<br></ul><br>An empty line separates the input data sets.</div><p class="pst">Output</p><div class="ptx" lang="en-US">For each data set the program must print (from the beginning of a line) the maximum profit for that set. The results must be printed on the standard output. An empty line separates the results of different data sets. An example is given in the following:</div><p class="pst">Sample Input</p><pre class="sio">3
2
2
4
4
1 2 14
3 4 25
2 4 12
3 3 10</pre><p class="pst">Sample Output</p><pre class="sio">26</pre><p class="pst">Source</p><div class="ptx" lang="en-US"><a href="searchproblem?field=source&key=Southeastern+Europe+2002">Southeastern Europe 2002</a></div></td></tr></table>
