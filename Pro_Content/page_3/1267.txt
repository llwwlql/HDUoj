<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 10000K</td></tr><tr><td><b>Total Submissions:</b> 813</td><td width="10px"></td><td><b>Accepted:</b> 194</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US">Workers are going to enclose a new working region with a fence. For their convenience the enclosed area has to be as large as possible. They have N rectangular blocks to build the fence. The length of the i-th block is Li meters. All blocks have the same height of 1 meter. The workers are not allowed to break blocks into parts. All blocks must be used to build the fence.</div><p class="pst">Input</p><div class="ptx" lang="en-US">The first line of the input file contains one integer N (3 <= N <= 100).<br>The following N lines describe fence blocks. S must be written with at least two digits after the decimal point. If it is not possible to construct the fence from the specified blocks, write 0.00.</div><p class="pst">Sample Input</p><pre class="sio">4
10
5
5
4
</pre><p class="pst">Sample Output</p><pre class="sio">28.00</pre><p class="pst">Source</p><div class="ptx" lang="en-US"><a href="searchproblem?field=source&key=Northeastern+Europe+2001">Northeastern Europe 2001</a>, Northern Subregion 2001(URAL 1159)</div></td></tr></table>
