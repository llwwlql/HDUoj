<div style='position: absolute; right: 10px;'>Language:<select size=1 onChange="window.location.href='problem?id=2832&lang='+this.value+'&change=true'"><option value='default' selected>Default</option></select></div><div class="ptt" lang="en-US">How Many Pairs?</div>
<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 3000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 65536K</td></tr><tr><td><b>Total Submissions:</b> 1858</td><td width="10px"></td><td><b>Accepted:</b> 674</td></tr><tr><td colspan="3" align="center" style="color:#0ead00"><b>Case Time Limit:</b> 1000MS</td></tr></table></div><p class="pst">Description</p><div class="ptx" lang="en-US"><p>You are given an undirected graph <i>G</i> with <i>N</i> vertices and <i>M</i> edges. Each edge has a length. Below are two definitions.</p><ol><li>Define <i>max_len</i>(<i>p</i>) as the length of the edge with the maximum length of <i>p</i> where <i> p</i> is an arbitrary non-empty path in <i>G</i>.</li><li>Define <i>min_pair</i>(<i>u</i>, <i>v</i>) as min{<i>max_len</i>(<i>p</i>) | <i>p</i> is a path connecting the vertices <i>u</i> and <i>v</i>.}. If there is no paths connecting <i>u</i> and <i>v</i>, <i>min_pair</i>(<i>u</i>, <i>v</i>) is defined as infinity.</li></ol><p>Your task is to count the number of (unordered) pairs of vertices <i>u</i> and <i>v</i> satisfying the condition that <i>min_pair</i>(<i>u</i>, <i>v</i>) is not greater than a given integer <i>A</i>.</p></div><p class="pst">Input</p><div class="ptx" lang="en-US"><p>The first line of input contains three integer <i>N</i>, <i>M</i> and <i>Q</i> (1 &lt; <i>N</i> &le; 10,000, 0 &lt; <i>M</i> &le; 50,000, 0 &lt; <i>Q</i> &le; 10,000). <i>N</i> is the number of vertices, <i>M</i> is the number of edges and <i>Q</i> is the number of queries. Each of the next <i>M</i> lines contains three integers <i>a</i>, <i>b</i>, and <i>c</i> (1 &le; <i>a</i>, <i>b</i> &le; <i>N</i>, 0 &le; <i>c</i> &lt; 10<sup>8</sup>) describing an edge connecting the vertices <i>a</i> and <i>b</i> with length <i>c</i>. Each of the following <i>Q</i> lines gives a query consisting of <i>a</i> single integer <i>A</i> (0 &le; <i>A</i> &lt; 10<sup>8</sup>).</p></div><p class="pst">Output</p><div class="ptx" lang="en-US"><p>Output the answer to each query on a separate line.</p></div><p class="pst">Sample Input</p><pre class="sio">4 5 4
1 2 1
2 3 2
2 3 5
3 4 3
4 1 4
0
1
3
2</pre><p class="pst">Sample Output</p><pre class="sio">0
1
6
3</pre><p class="pst">Source</p><div class="ptx" lang="en-US"><a href="searchproblem?field=source&key=POJ+Monthly--2006.05.28">POJ Monthly--2006.05.28</a>, zhucheng</div></td></tr></table>
