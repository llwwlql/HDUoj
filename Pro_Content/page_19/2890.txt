<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 131072K</td></tr><tr><td><b>Total Submissions:</b> 1671</td><td width="10px"></td><td><b>Accepted:</b> 518</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US"><p>Many studies have been done on developing efficient algorithms to calculate matrix multiplication. But it remains as a hard topic today. In this problem you are to calculate the 2006th power of a square Boolean matrix where addition and multiplication are defined as follows:</p><div align=center><table border="0"><tr><td><div align=center><table border="1" width="170"><tr><td align="center" width="30%"><i>A</i></td><td align="center" width="30%"><i>B</i></td><td align="center" width="40%"><i>A</i> + <i>B</i></td></tr><tr><td align="center" width="30%">0</td><td align="center" width="30%">0</td><td align="center" width="40%">0</td></tr><tr><td align="center" width="30%">0</td><td align="center" width="30%">1</td><td align="center" width="40%">1</td></tr><tr><td align="center" width="30%">1</td><td align="center" width="30%">0</td><td align="center" width="40%">1</td></tr><tr><td align="center" width="30%">1</td><td align="center" width="30%">1</td><td align="center" width="40%">1</td></tr></table><p align=center><b>Truth Table of Addition</b></p></div></td><td width="20"></td><td><div align=center><table border="1" width="170"><tr><td align="center" width="30%"><i>A</i></td><td align="center" width="30%"><i>B</i></td><td align="center" width="40%"><i>AB</i></td></tr><tr><td align="center" width="30%">0</td><td align="center" width="30%">0</td><td align="center" width="40%">0</td></tr><tr><td align="center" width="30%">0</td><td align="center" width="30%">1</td><td align="center" width="40%">0</td></tr><tr><td align="center" width="30%">1</td><td align="center" width="30%">0</td><td align="center" width="40%">0</td></tr><tr><td align="center" width="30%">1</td><td align="center" width="30%">1</td><td align="center" width="40%">1</td></tr></table><p align=center><b>Truth Table of Multiplication</b></p></div></td></tr></table></div><p>Let <i>A</i> be a square matrix. The zeroth power of <i>A</i> is the identity matrix. The <i>n</i>-th (<i>n</i> &gt; 0) power of <i>A</i> is the product of <i>A</i> and its (<i>n</i> <span lang="en-us">&minus;</span> 1)-th power.</p></div><p class="pst">Input</p><div class="ptx" lang="en-US"><p>The input contains multiple test cases. Each test cases consists of some lines:</p><ul><li>Line 1: Contains two integers <i>K</i> (<i>K</i> &lt; 1000) and <i>M</i> (0 <span lang="en-us">&le; <i>M</i> &le; 10<i>K</i>), indicating the dimension of the matrix is <i>K</i> &times; <i>K</i> and <i>K</i> + <i>M</i> elements of the matrix are 1&rsquo;s.</li><li>Lines 2 ~ <i>M</i> + 1: Each contains two integers <i>i</i> and <i>j</i> (0 &le; <i>i</i>, <i>j</i> &lt; <i>K</i>, <i>i</i> &ne; <i>j</i>), indicating the element in the (<i>i</i> + 1)-th row and (<i>j</i> + 1)-th column is 1.</li></ul><p>All elements on the primary diagonal of the matrix are 1&rsquo;</span>s.</p></div><p class="pst">Output</p><div class="ptx" lang="en-US"><p>For each test case output one line containing the number of elements that are 1<span lang="en-us">&rsquo;</span>s in the 2006th power of the given matrix.</p></div><p class="pst">Sample Input</p><pre class="sio">3 4
1 2
2 1
0 1
0 2</pre><p class="pst">Sample Output</p><pre class="sio">7</pre><p class="pst">Source</p><div class="ptx" lang="en-US"><a href="searchproblem?field=source&key=POJ+Monthly--2006.07.30">POJ Monthly--2006.07.30</a>, Static</div></td></tr></table>
