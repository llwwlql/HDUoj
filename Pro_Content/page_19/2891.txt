<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 131072K</td></tr><tr><td><b>Total Submissions:</b> 14494</td><td width="10px"></td><td><b>Accepted:</b> 4743</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US"></p>Elina is reading a book written by Rujia Liu, which introduces a strange way to express non-negative integers. The way is described as following:</p><p>Choose <i>k</i> different positive integers <i>a</i><sub>1</sub>, <i>a</i><sub>2</sub>, <span lang="en-us">&hellip;, <i>a<sub>k</sub></i>. For some non-negative <i>m</i>, divide it by every <i>a<sub>i</sub></i> (1 &le; <i>i</i> &le; <i>k</i>) to find the remainder <i>r<sub>i</sub></i>. If <i>a</i><sub>1</sub>, <i>a</i><sub>2</sub>, &hellip;, <i>a<sub>k</sub></i> are properly chosen, m can be determined, then the pairs (<i>a<sub>i</sub></i>, <i>r<sub>i</sub></i>) can be used to express <i>m</i>.</p><p>&ldquo;It is easy to calculate the pairs from <i>m</i>, &rdquo; said Elina. &ldquo;But how can I find <i>m</i> from the pairs?&rdquo;</span></p><p>Since Elina is new to programming, this problem is too difficult for her. Can you help her?</p></div><p class="pst">Input</p><div class="ptx" lang="en-US"><p>The input contains multiple test cases. Each test cases consists of some lines.</p><ul><li>Line 1: Contains the integer <i>k</i>.</li><li>Lines 2 ~ <i>k</i> + 1: Each contains a pair of integers <i>a<sub>i</sub></i>, <i>r<sub>i</sub></i> (1 &le; <i>i</i> &le;</span> <i>k</i>).</ul></p></div><p class="pst">Output</p><div class="ptx" lang="en-US"><p>Output the non-negative integer <i>m</i> on a separate line for each test case. If there are multiple possible values, output the smallest one. If there are no possible values, output <tt>-1</tt>.</p></p></div><p class="pst">Sample Input</p><pre class="sio">2
8 7
11 9</pre><p class="pst">Sample Output</p><pre class="sio">31</pre><p class="pst">Hint</p><div class="ptx" lang="en-US"><p>All integers in the input and the output are non-negative and can be represented by 64-bit integral types.</p></div><p class="pst">Source</p><div class="ptx" lang="en-US"><a href="searchproblem?field=source&key=POJ+Monthly--2006.07.30">POJ Monthly--2006.07.30</a>, Static</div></td></tr></table>
