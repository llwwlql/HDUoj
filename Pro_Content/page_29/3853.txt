<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 65536K</td></tr><tr><td><b>Total Submissions:</b> 729</td><td width="10px"></td><td><b>Accepted:</b> 184</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US">Ethan wants to draw a painting on an m×n board. He can draw some strips on the board using a paintbrush of width one. In each step, he must choose a new color and paint a full column or a full row.<br>He has a great image to be drawn on the board, but he doesn’t know which color to use first. You must help him in finding out the order of colors.<br><center><img src=images/3853_1.png></center></div><p class="pst">Input</p><div class="ptx" lang="en-US">There are multiple test cases in the input. The first line of each test case contains two integers m and n, the size of the board (0< m, n <100). Following the first line, there are m lines with n integers denoting the color in each cell. All the colors are positive integer numbers less than 10000. The input is terminated with a single line containing two consecutive zeros.</div><p class="pst">Output</p><div class="ptx" lang="en-US">For each test case, write a single line containing the order of colors used to paint the board. If there are several answers, output the one which is lexicographically smallest (considering each number as a symbol).</div><p class="pst">Sample Input</p><pre class="sio">4 4
1 5 4 3
6 5 6 6
2 2 2 2
