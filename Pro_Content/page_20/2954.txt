<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 65536K</td></tr><tr><td><b>Total Submissions:</b> 5827</td><td width="10px"></td><td><b>Accepted:</b> 2504</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US"><p>A <i>lattice point</i> is an ordered pair (<i>x</i>, <i>y</i>) where <i>x</i> and <i>y</i> are both integers. Given the coordinates of the vertices of a triangle (which happen to be lattice points), you are to count the number of lattice points which lie completely inside of the triangle (points on the edges or vertices of the triangle do not count).</p></div><p class="pst">Input</p><div class="ptx" lang="en-US"><span lang="en-us"><p>The input test file will contain multiple test cases. Each input test case consists of six integers <i>x</i><sub>1</sub>, <i>y</i><sub>1</sub>, <i>x</i><sub>2</sub>, <i>y</i><sub>2</sub>, <i>x</i><sub>3</sub>, and <i>y</i><sub>3</sub>, where (<i>x</i><sub>1</sub>, <i>y</i><sub>1</sub>), (<i>x</i><sub>2</sub>, <i>y</i><sub>2</sub>), and (<i>x</i><sub>3</sub>, <i>y</i><sub>3</sub>) are the coordinates of vertices of the triangle. All triangles in the input will be non-degenerate (will have positive area), and &minus;15000 &le; <i>x</i><sub>1</sub>, <i>y</i><sub>1</sub>, <i>x</i><sub>2</sub>, <i>y</i><sub>2</sub>, <i>x</i><sub>3</sub>, <i>y</i><sub>3</sub> &le; 15000. The end-of-file is marked by a test case with <i>x</i><sub>1</sub> =&nbsp; <i>y</i><sub>1</sub> = <i>x</i><sub>2</sub> = <i>y</i><sub>2</sub> = <i>x</i><sub>3</sub> = <i>y</i><sub>3</sub> = 0 and should not be processed.</p></span></div><p class="pst">Output</p><div class="ptx" lang="en-US"><p>For each input case, the program should print the number of internal lattice points on a single line.</p></div><p class="pst">Sample Input</p><pre class="sio">0 0 1 0 0 1
0 0 5 0 0 5
0 0 0 0 0 0</pre><p class="pst">Sample Output</p><pre class="sio">0
6</pre><p class="pst">Source</p><div class="ptx" lang="en-US"><a href="searchproblem?field=source&key=Stanford+Local+2004">Stanford Local 2004</a></div></td></tr></table>
