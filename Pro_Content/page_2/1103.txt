<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 10000K</td></tr><tr><td><b>Total Submissions:</b> 1253</td><td width="10px"></td><td><b>Accepted:</b> 499</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US">By filling a rectangle with slashes (/) and backslashes (\), you can generate nice little mazes. Here is an example: <br><center><img src="images/1103/1103_1.gif"></center><br>As you can see, paths in the maze cannot branch, so the whole maze only contains cyclic paths and paths entering somewhere and leaving somewhere else. We are only interested in the cycles. In our example, there are two of them. <br><br>Your task is to write a program that counts the cycles and finds the length of the longest one. The length is defined as the number of small squares the cycle consists of (the ones bordered by gray lines in the picture). In this example, the long cycle has length 16 and the short one length 4. <br></div><p class="pst">Input</p><div class="ptx" lang="en-US">The input contains several maze descriptions. Each description begins with one line containing two integers w and h (1 <= w,h <= 75), the width and the height of the maze. The next h lines represent the maze itself, and contain w characters each; all these characters will be either "/" or "\". <br><br>The input is terminated by a test case beginning with w = h = 0. This case should not be processed. </div><p class="pst">Output</p><div class="ptx" lang="en-US">For each maze, first output the line "Maze #n:", where n is the number of the maze. Then, output the line "k Cycles; the longest has length l.", where k is the number of cycles in the maze and l the length of the longest of the cycles. If the maze does not contain any cycles, output the line "There are no cycles.". <br><br>Output a blank line after each test case. </div><p class="pst">Sample Input</p><pre class="sio">6 4
\//\\/
\///\/
//\\/\
\/\///
3 3
///
\//
\\\
0 0</pre><p class="pst">Sample Output</p><pre class="sio">Maze #1:
2 Cycles; the longest has length 16.

Maze #2:
There are no cycles.</pre><p class="pst">Source</p><div class="ptx" lang="en-US"><a href="searchproblem?field=source&key=Mid-Central+European+Regional+Contest+1999">Mid-Central European Regional Contest 1999</a></div></td></tr></table>
