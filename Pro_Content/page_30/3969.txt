<div style='position: absolute; right: 10px;'>Language:<select size=1 onChange="window.location.href='problem?id=3969&lang='+this.value+'&change=true'"><option value='default' selected>Default</option></select></div><div class="ptt" lang="en-US">K-Graph Oddity</div>
<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 2500MS</td><td width="10px"></td><td colspan="3"><b>Memory Limit:</b> 65536K</td></tr><tr><td><b>Total Submissions:</b> 684</td><td width="10px"></td><td><b>Accepted:</b> 101</td><td width="10px"></td><td style="font-weight:bold; color:red;">Special Judge</td></tr></table></div><p class="pst">Description</p><div class="ptx" lang="en-US">You are given a connected undirected graph with an odd number of vertices. The degree of the vertex, by definition, is the number of edges incident to it. In the given graph the degree of each vertex does not exceed an odd number <i>k</i>. Your task is to color the vertices of this graph into at most <i>k</i> distinct colors, so that the colors of any two adjacent vertices are distinct.<br>The pictures below show two graphs. The first one has 3 vertices and the second one has 7 vertices. In both graphs degrees of the vertices do not exceed 3 and the vertices are colored into at most 3 different colors marked as '<img src="images/3969_1.png">', '<img src="images/3969_2.png">', and '<img src="images/3969_3.png">'.<br><br><center><img src="images/3969_4.png"></center></div><p class="pst">Input</p><div class="ptx" lang="en-US">The first line of the input file contains two integer numbers <i>n</i> and <i>m</i>, where <i>n</i> is the number of vertices in the graph (3 &lt;= <i>n</i> &lt;= 9999, <i>n</i> is odd), <i>m</i> is the number of edges in the graph (2 &lt;= m &lt;= 100 000). The following <i>m</i> lines describe edges of the graph, each edge is described by two integers <i>a<sub>i</sub></i>, <i>b<sub>i</sub></i> (1 &lt;= <i>a<sub>i</sub></i>, <i>b<sub>i</sub></i> &lt;= n, <i>a<sub>i</sub></i> != <i>b<sub>i</sub></i>) — the vertex numbers connected by this edge. Each edge is listed at most once. The graph in the input file is connected, so there is a path between any pair of vertices.</div><p class="pst">Output</p><div class="ptx" lang="en-US">On the first line of the output file write a single integer number <i>k</i> — the minimal odd integer number, such that the degree of any vertex does not exceed <i>k</i>. Then write <i>n</i> lines with one integer number <i>c<sub>i</sub></i> (1 &lt;= <i>c<sub>i</sub></i> &lt;= <i>k</i>) on a line that denotes the color of <i>i</i>-th vertex.<br>The colors of any two adjacent vertices must be different. If the graph has multiple different colorings, print any of them. At least one such coloring always exists.</div><p class="pst">Sample Input</p><pre class="sio"><table border="1" style="border-collapse: collapse" bordercolor="#000000" id="table1"><tr>
