<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 10000MS</td><td width="10px"></td><td colspan="3"><b>Memory Limit:</b> 65536K</td></tr><tr><td><b>Total Submissions:</b> 888</td><td width="10px"></td><td><b>Accepted:</b> 203</td><td width="10px"></td><td style="font-weight:bold; color:red;">Special Judge</td></tr></table></div><p class="pst">Description</p><div class="ptx" lang="en-US">Flatland government is building a new highway that will be used to transport weapons from its main weapon plant to the frontline in order to support the undergoing military operation against its neighbor country Edgeland. Highway is a straight line and there are <i>n</i> construction teams working at some points on it.<br>During last days the threat of a nuclear attack from Edgeland has significantly increased. Therefore the construction office has decided to develop an evacuation plan for the construction teams in case of a nuclear attack. There are <i>m</i> shelters located near the constructed highway. This evacuation plan must assign each team to a shelter that it should use in case of an attack.<br>Each shelter entrance must be securely locked from the inside to prevent any damage to the shelter itself. So, for each shelter there must be some team that goes to this shelter in case of an attack. The office must also supply fuel to each team, so that it can drive to its assigned shelter in case of an attack. The amount of fuel that is needed is proportional to the distance from the team’s location to the assigned shelter. To minimize evacuation costs, the office would like to create a plan that minimizes the total fuel needed.<br/>Your task is to help them develop such a plan.</div><p class="pst">Input</p><div class="ptx" lang="en-US">The first line of the input file contains <i>n</i> — the number of construction teams (1 &lt;= <i>n</i> &lt;= 4000). The second line contains <i>n</i> integer numbers — the locations of the teams. Each team’s location is a positive integer not exceeding 10<sup>9</sup>, all team locations are different.<br>The third line of the input file contains <i>m</i> — the number of shelters (1 &lt;= <i>m</i> &lt;= <i>n</i>). The fourth linecontains <i>m</i> integer numbers — the locations of the shelters. Each shelter’s location is a positive integer not exceeding 10<sup>9</sup>, all shelter locations are different.<br>The amount of fuel that needs to be supplied to a team at location x that goes to a shelter at location y is equal to |x-y|.</div><p class="pst">Output</p><div class="ptx" lang="en-US">The first line of the output file must contain <i>z</i> — the total amount of fuel needed. The second line must contain <i>n</i> integer numbers: for each team output the number of the shelter that it should be assigned to.<br>Shelters are numbered from 1 to <i>m</i> in the order they are listed in the input file.</div><p class="pst">Sample Input</p><pre class="sio">3
1 2 3
2
2 10</pre><p class="pst">Sample Output</p><pre class="sio">8
1 1 2</pre><p class="pst">Source</p><div class="ptx" lang="en-US"><a href="searchproblem?field=source&key=Northeastern+Europe+2010">Northeastern Europe 2010</a></div></td></tr></table>
