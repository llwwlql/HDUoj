<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 65536K</td></tr><tr><td><b>Total Submissions:</b> 494</td><td width="10px"></td><td><b>Accepted:</b> 246</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US">We say that a set S = {x1, x2, ..., xn} is factor closed if for any xi ∈ S and any divisor d of xi we have d ∈ S. Let’s build a GCD matrix (S) = (sij), where sij = GCD(xi, xj) – the greatest common divisor of xi and xj. Given the factor closed set S, find the value of the determinant:<br><br><center><img src="formula?tex=D_n+%3D+%5Cleft%7C%7B%5Cbegin%7Barray%7D%7Bccccc%7Dgcd%28x_1%2Cx_1%29%26gcd%28x_1%2Cx_2%29%26gcd%28x_1%2Cx_3%29%26%5Ccdots%26gcd%28x_1%2Cx_n%29%5C%5Cgcd%28x_2%2Cx_1%29%26gcd%28x_2%2Cx_2%29%26gcd%28x_2%2Cx_3%29%26%5Ccdots%26gcd%28x_2%2Cx_n%29%5C%5Cgcd%28x_3%2Cx_1%29%26gcd%28x_3%2Cx_2%29%26gcd%28x_3%2Cx_3%29%26%5Ccdots%26gcd%28x_3%2Cx_n%29%5C%5C%5Ccdots%26%5Ccdots%26%5Ccdots%26%5Ccdots%26%5Ccdots%5C%5Cgcd%28x_n%2Cx_1%29%26gcd%28x_n%2Cx_2%29%26gcd%28x_n%2Cx_3%29%26%5Ccdots%26gcd%28x_n%2Cx_n%29%5Cend%7Barray%7D%7D%5Cright%7C" alt="D_n = \left|{\begin{array}{ccccc}gcd(x_1,x_1)&#38;gcd(x_1,x_2)&#38;gcd(x_1,x_3)&#38;\cdots&#38;gcd(x_1,x_n)\\gcd(x_2,x_1)&#38;gcd(x_2,x_2)&#38;gcd(x_2,x_3)&#38;\cdots&#38;gcd(x_2,x_n)\\gcd(x_3,x_1)&#38;gcd(x_3,x_2)&#38;gcd(x_3,x_3)&#38;\cdots&#38;gcd(x_3,x_n)\\\cdots&#38;\cdots&#38;\cdots&#38;\cdots&#38;\cdots\\gcd(x_n,x_1)&#38;gcd(x_n,x_2)&#38;gcd(x_n,x_3)&#38;\cdots&#38;gcd(x_n,x_n)\end{array}}\right|" align="absmiddle"></center></div><p class="pst">Input</p><div class="ptx" lang="en-US">The input file contains several test cases. Each test case starts with an integer n (0 < n < 1000), that stands for the cardinality of S. The next line contains the numbers of S: x1, x2, ..., xn. It is known that each xi is an integer, 0 < xi < 2*10<sup>9</sup>. The input data set is correct and ends with an end of file.</div><p class="pst">Output</p><div class="ptx" lang="en-US">For each test case find and print the value Dn mod 1000000007.</div><p class="pst">Sample Input</p><pre class="sio">2 
1 2 
3 
1 3 9 
4 
1 2 3 6</pre><p class="pst">Sample Output</p><pre class="sio">1 
12 
4</pre><p class="pst">Source</p><div class="ptx" lang="en-US"><a href="searchproblem?field=source&key=Southeastern+European+Regional+Programming+Contest+2008">Southeastern European Regional Programming Contest 2008</a></div></td></tr></table>
