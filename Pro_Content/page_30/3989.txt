<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 65536K</td></tr><tr><td><b>Total Submissions:</b> 489</td><td width="10px"></td><td><b>Accepted:</b> 244</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US">Math Olympiad is called “Aoshu” in China. Aoshu is very popular in elementary schools. Nowadays, Aoshu is getting more and more difficult. Here is a classic Aoshu problem:<br><br><center>ABBDE __ ABCCC = BDBDE</center><br><br>In the equation above, a letter stands for a digit(0 – 9), and different letters stands for different digits. After you fill the blank with '+', '-', '×' or '÷', and replace the letters with digits, you get an equation.<br><br>How to make the equation right? Here is a solution:<br><br><center>12245 + 12000 = 24245</center><br><br>In that solution, A = 1, B = 2, C = 0, D = 4, E = 5, and '+' is filled in the blank.<br><br>When I was a kid, finding a solution is OK. But now, my daughter’s teacher tells her to find all solutions. That’s terrible. I doubt whether her teacher really knows how many solutions are there. So please write a program for me to solve this kind of problems.</div><p class="pst">Input</p><div class="ptx" lang="en-US">The first line of the input is an integer T( T <= 20) indicating the number of test cases.<br><br>Each test case is a line which is in the format below:<br><br>s1 s2 s3 <br><br>s1, s2 and s3 are all strings which are made up of capital letters. Those capital letters only include 'A','B','C','D' and 'E', so forget about 'F' to 'Z'. The length of s1,s2 or s3 is no more than 8.<br><br>When you put a '=' between s2 and s3, and put a operator('+', '-', '×' or '÷'.) between s1 and s2, and replace every capital letter with a digit, you get a equation. <br>You should figure out the number of solutions making the equation right.<br><br>Please note that same letters must be replaced by same digits, and different letters must be replaced by different digits.  If a number in the equation is more than one digit, it must not have leading zero.</div><p class="pst">Output</p><div class="ptx" lang="en-US">For each test case, print an integer in a line. It represents the number of solutions.</div><p class="pst">Sample Input</p><pre class="sio">2
A A A
BCD BCD B
</pre><p class="pst">Sample Output</p><pre class="sio">5
72
</pre><p class="pst">Source</p><div class="ptx" lang="en-US"><a href="searchproblem?field=source&key="></a></div></td></tr></table>
