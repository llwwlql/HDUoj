<div class="plm"><table align="center"><tr><td><b>Time Limit:</b> 1000MS</td><td width="10px"></td><td><b>Memory Limit:</b> 65536K</td></tr><tr><td><b>Total Submissions:</b> 915</td><td width="10px"></td><td><b>Accepted:</b> 356</td></table></div><p class="pst">Description</p><div class="ptx" lang="en-US"><img src=images/2639_1.jpg align=right>The set of cyclic rotations of a string are the strings obtained by embedding the string clockwise on a ring, with the first character following on the last, starting at any character position and moving clockwise on the ring until the character preceeding the starting character is reached. A string is a necklace if it is the lexicographically smallest among all its cyclic rotations. For instance, for the string 01011 the cyclic rotations are (10110,01101,11010,10101,01011), and furthermore 01011 is the smallest string and hence, a necklace.<br>Any string S can be written in a unique way as a concatenation S = T1T2 . . . Tk of necklaces Ti such that T<sub>i+1</sub> < Ti for all i = 1, . . . , k - 1, and TiT<sub>i+1</sub> is not a necklace for any i = 1, . . . , k - 1. This representation is called the necklace decomposition of the string S, and your task is to find it.<br>The relation < on two strings is the lexicographical order and has the usual interpretation: A < B if A is a proper prefix of B or if A is equal to B in the first j - 1 positions but smaller in the jth position for some j. For instance, 001 < 0010 and 1101011 < 1101100.</div><p class="pst">Input</p><div class="ptx" lang="en-US">On the first line of the input is a single positive integer n, telling the number of test scenarios to follow. Each scenario consists of one line containing a non-empty string of zeros and ones of length at most 100.</div><p class="pst">Output</p><div class="ptx" lang="en-US">For each scenario, output one line containing the necklace decomposition of the string. The necklaces should be written as '(' necklace ')'.</div><p class="pst">Sample Input</p><pre class="sio">5
0
0101
0001
0010
11101111011</pre><p class="pst">Sample Output</p><pre class="sio">(0)
(0101)
(0001)
(001)(0)
(111)(01111)(011)</pre><p class="pst">Source</p><div class="ptx" lang="en-US"><a href="searchproblem?field=source&key=Nordic+2005">Nordic 2005</a></div></td></tr></table>
